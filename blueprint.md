# 프로젝트 청사진: 다재다능 계산기 Hub

## 프로젝트 개요
'다재다능 계산기 Hub'는 사용자의 생활, 금융, 업무, 건강 등 다양한 분야의 계산을 돕는 웹 애플리케이션입니다. 프레임워크 없이 순수 HTML, CSS, JavaScript(ES Modules)와 Web Components를 활용하여 구축됩니다.

## 현재 구현된 기능 및 디자인
- **UI/UX**: 
  - 사이드바 카테고리 필터링 (금융, 세금, 업무, 생활, 건강).
  - 상단 검색 바를 통한 실시간 검색.
  - 그리드 레이아웃 기반의 계산기 카드 시스템.
  - 개별 계산기 상세 보기 및 결과 출력.
- **인증 시스템**:
  - Firebase Authentication을 통한 Google 로그인 연동.
  - 로그인 상태에 따른 헤더 UI 동적 변경 (프로필 이미지, 이름 표시).
  - 로그아웃 기능 구현.
- **계산기 목록 (2026 최신 데이터 반영)**:
  - 주식 배당금 계산기
  - 연봉/실수령액 계산기 (2026년 국민연금 4.75%, 건강보험 3.595% 등 반영)
  - 시급 ↔ 월급 변환기 (2026년 최저임금 10,320원 반영)
  - 대출 이자/상환 계산기
  - 유류비 계산기
  - 퍼센트/증감율 계산기
  - BMI 비만도 계산기

## 최근 작업 내용 (2026.03.31)
- **AdSense 심사 대응 및 콘텐츠 고도화**:
  - **필수 법적 문서 추가**: `privacy.html`(개인정보처리방침), `terms.html`(이용약관) 페이지 신규 구축.
  - **E-A-T 강화**: `about.html`(서비스 소개) 페이지 추가 및 운영자 연락처 명시.
  - **저품질 콘텐츠 탈피**: 메인 페이지 하단에 각 계산기별 산출 공식(Salary, Loan, BMI 등) 및 상세 이용 가이드(FAQ) 섹션 대폭 보강.
  - **네비게이션 개선**: 푸터에 필수 페이지 링크 연결 및 스타일 최적화.
  - **SEO 최적화**: 신규 페이지를 반영한 `sitemap.xml` 갱신 및 구조화 데이터 최적화 상태 점검.

## 향후 계획
- **사용자 맞춤형 기능**: 로그인한 사용자를 위한 계산 기록 저장 기능.
- **공유 기능**: 계산 결과를 카카오톡이나 링크로 공유하는 기능.
- **추가 계산기**: 부동산 취득세, 적금 만기 수령액 등 추가.
