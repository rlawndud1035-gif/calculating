# 프로젝트 청사진: 다재다능 계산기 Hub

## 프로젝트 개요
'다재다능 계산기 Hub'는 사용자의 생활, 금융, 업무, 건강 등 다양한 분야의 계산을 돕는 웹 애플리케이션입니다. 프레임워크 없이 순수 HTML, CSS, JavaScript(ES Modules)와 Web Components를 활용하여 구축됩니다.

## 현재 구현된 기능 및 디자인
- **UI/UX**: 
  - 사이드바 카테고리 필터링 (금융, 세금, 업무, 생활, 건강).
  - 상단 검색 바를 통한 실시간 검색.
  - 그리드 레이아웃 기반의 계산기 카드 시스템.
  - 개별 계산기 상세 보기 및 결과 출력.
- **계산기 목록**:
  - 주식 배당금 계산기
  - 연봉/실수령액 계산기
  - 시급 ↔ 월급 변환기
  - 대출 이자/상환 계산기
  - 유류비 계산기
  - 퍼센트/증감율 계산기
  - BMI 비만도 계산기

## 향후 계획: 구글 로그인 연동 (현재 작업 중)
Firebase Authentication을 사용하여 사용자가 구글 계정으로 로그인할 수 있는 기능을 추가합니다.

### 단계별 계획
1. **환경 설정**:
   - `.idx/mcp.json`에 Firebase MCP 설정 추가.
   - `index.html`에 Firebase SDK (App, Auth) 추가.
2. **UI 수정**:
   - 헤더 우측에 'Google로 로그인' 버튼 추가.
   - 로그인 후 사용자 프로필 이미지 및 '로그아웃' 버튼 표시.
3. **인증 로직 구현**:
   - Firebase 앱 초기화.
   - Google Auth Provider 설정.
   - `signInWithPopup`을 이용한 로그인 구현.
   - `onAuthStateChanged`를 이용한 로그인 상태 유지 및 UI 동기화.
4. **검증**:
   - 로그인/로그아웃 동작 확인.
   - 사용자 정보(이름 등)가 정상적으로 표시되는지 확인.
